Everything: Mark the part of the readme you want to take with the field name, e.g. <!-- Description --> <!-- /Description -->

Title: On the title line as such: <!-- Title --> *Title* <!-- /Title -->

Description: See above

Git link: If git link is changed the webpage will not detect that

Authors: The authors to be used must be at the top, else the readme will have a space where the authors tag is inserted

Keywords: Add keywords in - [item] format between <!-- Keyword --> <!-- /Keyword --> tag, no extra spaces and keywords should be separated with \n
